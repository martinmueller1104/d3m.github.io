## In preparation for WS 2026/27 and SS 2027

To offer students an easily accessible starting point for the topics of data and machine learning in materials science, we are currently setting up three new hands-on experiments:
- Data-Driven Materials Selection: select suitable materials for a specific application based on a materials database and resolve conflicting objectives
- Machine learning with tabular data: apply forward modeling and inverse design on an examplary materials science dataset
- Machine learning with images: train a U-Net model for microstructure segmentation
